# IP Addressing

## What is an IP Address?
An **IP (Internet Protocol) address** is a **logical address** used to identify devices on a network.
It allows devices to **send and receive data across networks**.


## Types of IP Addresses

### 1. IPv4
- 32-bit address
- Example:192.168.1.1

  
### 2. IPv6
- 128-bit address
- Example:2001:db8::1


## Private vs Public IP

### Private IP
- Used inside local networks
- Example:192.168.x.x


### Public IP
- Assigned by ISP (Internet Service Provider)
- Used on the internet
  

## Cybersecurity Importance
- IP spoofing attacks
- IP tracking and blocking
- Firewall rules based on IP
