# Transport Layer (Layer 4)

The Transport Layer is responsible for **end-to-end communication** between applications.
It ensures data is delivered **reliably, correctly, and in order**.

## Why Transport Layer is Important in Cybersecurity

- Controls **ports and sessions**
- Used in **DoS/DDoS attacks**
- Targeted by **SYN flood, port scanning**
- Essential for **firewall and IDS rules**
